# Sumi

A dark sakura theme for [Obsidian](https://obsidian.md) — ink-dark surfaces, sakura-pink accents, bold Zen typography, and falling petals.

![Sumi](screenshot.png)

## Features

- **Sumi-e palette** — ink-dark window, chrome and card surfaces with soft sakura-pink accents.
- **Bundled Zen fonts** — Zen Maru Gothic and Zen Kaku Gothic New ship as local `.woff2` files, so the theme makes no network requests and renders offline.
- **Falling petals** — a pure-CSS petal field drifts over the workspace, and it respects `prefers-reduced-motion` (no animation for readers who ask for none).
- **Style Settings toggles** — when the [Style Settings](https://github.com/mgmeyers/obsidian-style-settings) plugin is installed, you can change the accent colour, turn the petals off, switch heading weight, and remove card borders.

## Install

Manual install:

1. Copy the `Sumi` folder into `<your-vault>/.obsidian/themes/Sumi/`.
2. In Obsidian, open **Settings → Appearance → Themes** and select **Sumi**.

The theme may also come to Obsidian's community theme directory later, at which point it will be installable from **Settings → Appearance → Manage** without copying files by hand.

## Dark mode required

Sumi is a dark-only theme. Set **Settings → Appearance → Base color scheme** to **Dark**.
